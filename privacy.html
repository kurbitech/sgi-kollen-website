Integritetspolicy

Din integritet är viktig för oss

Tillbaka

Senast uppdaterad: 14 november 2025

1. Ingen personlig datainsamling

Vi samlar INTE in personuppgifter

All information du anger (inkomst, datum, val) sparas endast lokalt på din enhet. Ingen personlig data skickas till oss.

Vi samlar inte in:

  • • Namn, personnummer eller kontaktuppgifter
  • • Platsdata eller användarmönster
  • • Din inkomst eller ekonomiska information

2. Teknisk felsökning (Crash Analytics)

För att förbättra appens stabilitet kan vi samla in anonymiserad teknisk information om krascher:

  • • Enhetsmodell och iOS-version
  • • Tidpunkt för krasch
  • • Teknisk information om felet

Detta inkluderar INTE:

  • • Din inkomst eller andra siffror du angett
  • • Personuppgifter eller identifierare
  • • Information om hur du använder appen

Denna data används endast för att hitta och fixa buggar.

3. Lokal datalagring

Följande information sparas endast på din enhet:

  • • Din angivna månadsinkomst
  • • Barnets ålder
  • • Valda datum för föräldraledighet
  • • Dina markeringar i SGI-kalendern
  • • Dina inställningar och preferenser
  • • Acceptans av användarvillkor

Denna data:

  • • Lagras i appens lokala lagring (UserDefaults/lokala filer)
  • • Raderas automatiskt om du tar bort appen
  • • Synkroniseras INTE via iCloud
  • • Skickas ALDRIG till externa servrar

4. Apple App Store

App Store hanteras av Apple som kan samla in information om nedladdning, köp och recensioner. Se Apples integritetspolicy:

apple.com/legal/privacy
